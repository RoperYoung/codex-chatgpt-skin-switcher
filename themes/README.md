# Bundled themes

These are the sixteen theme packages bundled with ChatGPT Skin Switcher and approved by the project owner for presentation in this showcase repository.

Each directory contains:

- `home.jpg` — ChatGPT/home scene background
- `task.jpg` — Codex/task scene background
- `preview.jpg` — theme-market preview
- `manifest.json` — Manifest V1 metadata, tuning, hashes, and rights notice
