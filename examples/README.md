# Reference code

This directory contains three selected, non-core Swift reference files:

- `ThemePackageModels.swift` — Manifest V1 and catalog wire models
- `ThemeMarketModels.swift` — local/remote market item model
- `AtomicFileWriter.swift` — generic atomic JSON/file writer with symlink checks

They are provided to document selected data contracts and implementation patterns. They are not a complete SDK, do not include the production runtime or UI, and are not sufficient to build ChatGPT Skin Switcher.
